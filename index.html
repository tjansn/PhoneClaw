But Still...

A 20-Step Checklist

  • ! Install 12 Termux packages
  • ! Configure TMPDIR, TMP, TEMP
  • ! Node-GYP workaround
  • ! npm install openclaw
  • ! Find and patch ~15 files
  • ! Create service definition
  • ! Set up runit directories
  • ! Configure bashrc exports

Get one wrong and something downstream breaks in a confusing way.

Miss the GYP workaround?
Good luck with that error.

Forget one env var?
Permission denied.

Run setup twice?
PATH duplicated six times.

Improvement #2

I Sat Down with Claude

CLAUDE CODE
# The brief:
"Here's everything I do manually.
 Make it a script.
 Make it idempotent.
 Make it detect Termux.
 Make it patch every file automatically.
 Make it print clear instructions at the end."

# What Claude caught that I missed:
> bashrc block needs markers
> so re-running won't add duplicate lines
> otherwise PATH gets same dir 6x

We went back and forth. I'd test on the phone, find an edge case, come back. One evening.
